# Gallery Pipeline

1. `pip install -r scripts/requirements.txt`
2. Skopíruj fotky do `raw/` ako `NN-before.jpg` a `NN-after.jpg`.
3. Doplň riadok do `raw/gallery.csv`.
4. Spusť `npm run gallery`, potom commitni a pushni výstup.

Poznámka: prvé spustenie stiahne model približne 900 MB. Windows PowerShell je v poriadku, ak je `python` na `PATH`.
